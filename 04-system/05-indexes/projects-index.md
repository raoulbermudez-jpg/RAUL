# Projects Index — /RAUL/

Índice central de proyectos activos y recientes del sistema /RAUL/.

## Propósito

Este archivo es la referencia rápida para ubicar proyectos por dominio, estado y ruta.  
No sustituye el `README.md` dentro de cada proyecto; solo funciona como mapa central.

## Convención de estados

- `00-brief/` — intake inicial, brief crudo, material de entrada.
- `01-strategy-and-design/` — framing, estrategia, diseño conceptual.
- `02-production/` — producción activa de piezas, scripts, artes, borradores.
- `03-review-and-release/` — revisión, feedback, aprobaciones, iteraciones finales.
- `04-published-and-hand-off/` — entregado, publicado o handoff final.
