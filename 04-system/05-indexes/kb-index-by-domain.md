# KB Index by Domain — /RAUL/

Índice del contenido real dentro de `02-knowledge-base/` organizado por dominio. Complementa `domains-index.md` (que mapea estado de activación) con un inventario de qué existe en cada dominio.

## Propósito

Este archivo permite saber rápidamente:
- qué secciones de la KB de un dominio están pobladas vs. vacías,
- cuántos documentos hay en `specs/` por dominio,
- qué artículos existen en `wiki/`,
- y dónde están los assets.

No reemplaza los `_index.md` de cada dominio — esos son la puerta de entrada. Este índice es una vista transversal de alto nivel.

## Regla de mantenimiento

Actualizar cuando:
- se añaden documentos nuevos a `specs/` de cualquier dominio,
- se crean artículos en `wiki/`,
- un dominio pasa de stub a poblado,
- se incorporan assets nuevos.
