# 06-companion-docs/

Documentos auxiliares de apoyo a continuidad y diseño del sistema /RAUL/.

## Propósito

Esta carpeta concentra los documentos **companion**: material que apoya la operación o documenta sesiones de diseño, pero **no es Single Source of Truth** del sistema.

## Regla de estatus

Los documentos aquí almacenados:

- preservan contexto entre sesiones largas,
- resumen trabajo realizado,
- registran memoria operativa o blueprints,
- mantienen bibliotecas de consulta.

**No son autoridad final** sobre arquitectura, reglas o gobernanza. Para SSOT, ver:

- `04-system/01-config/` (FOLDER-ARCHITECTURE, CLAUDE_core, CONTEXT_core, NAMING-CONVENTIONS, LLM-GUIDELINES)
- `04-system/03-governance/` (DECISIONS, MIGRATION-PLAN, RISK-POLICY, SECURITY-AND-ACCESS)
- `04-system/05-indexes/` (índices operativos)

## Convención de nombres

Archivos terminados en ` PERPLEXITY.md` indican origen en sesiones de Perplexity (diseño asistido). Otros companion docs no requieren sufijo.
