# GRN-MV - Relé de Nivel para Líquidos Conductores

## Descripción General

El GRN-MV es un relé electrónico diseñado para controlar el vaciado o llenado de tanques con líquidos conductores. El control de los niveles se realiza mediante la conexión a electrodos de máximo nivel y/o de mínimo nivel (electrodos existentes en la instalación o provistos por el usuario).

El GRN-MV dispone de una salida de relé para encender automáticamente (y cada vez que se requiera) el circuito arrancador de una bomba eléctrica.

El GRN-MV contiene dos indicadores luminosos que reportan cuando hay alimentación eléctrica y cuando el relé de salida se encuentra activado.

## Configuración

- Perilla de ajuste, para fijar sensibilidad de los electrodos de apagado y encendido, ajuste de fábrica 10K
- Conexión para controlar dos niveles, puntos máximo y mínimo
- Conexión para controlar un nivel fijo

## Aplicaciones

- Tableros y bombas eléctricas, que suministran o descargan líquidos en tanques de almacenamiento
