# Protector de Voltaje para Compresores Monofásicos

## Características Principales

- **Capacidad**: Hasta 18.000 BTU
- **Voltaje**: 120 V~
- **Tipo de conexión**: Monofásico

## Protecciones

El dispositivo proporciona protección contra:

- Altos y bajos voltaje
- Inestabilidad de voltaje
- Parpadeos (fluctuaciones)
- Variaciones extremas

## Diagrama Eléctrico GSM-RF120B

### Conexiones

**Entrada:**
- F: Fase
- N: Neutro
- T: Tierra

**Salida:**
- F: Fase
- N: Neutro
- T: Tierra
- S: Salida hacia equipo a proteger
