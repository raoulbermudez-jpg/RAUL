# Fotocontrol Temporizado para Encendido y Apagado Automático de Iluminación Externa

## Descripción del Producto

Fotocontrol temporizado diseñado para automatizar el encendido y apagado de sistemas de iluminación externa. Producto manufacturado bajo rigurosas normas de control de calidad.

## Características Principales

- Máxima precisión
- Ahorro de energía
- Disminuye falsos apagados
- Comodidad
- Seguridad
- Para todo tipo de luminarias
- Fácil instalación
- Multivoltaje
- Altamente resistente
- Para ser instalado en exteriores
- Temporizado de dos (2) segundos de respuesta ante la luz

## Especificaciones Técnicas

**Modelo:** GC-33A-VE-V1

**Versión:** V1

**Temporización:** 2 segundos

**Instalación:** Exterior - Se debe instalar con la ventanilla mirando hacia el norte para optimizar el funcionamiento

## Instrucciones de Instalación

Para optimizar el funcionamiento, la fotocelda debe montarse con la ventanilla completamente abierta hacia el norte, simulando luz y oscuridad. Compruebe la correcta operación cubriendo y descubriendo la fotocelda durante más de dos (2) segundos (modelo GC-33A-VE-V1).
