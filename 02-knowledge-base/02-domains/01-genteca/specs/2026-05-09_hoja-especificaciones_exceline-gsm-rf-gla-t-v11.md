# Protector de Voltaje para Compresores Monofásicos

## Características Principales

- Protección contra altos y bajos voltaje
- Protección contra inestabilidad de voltaje
- Protección contra parpadeos
- Bornera para cables hasta AWG 8
- Fácil instalación

## Especificaciones Técnicas

### Capacidades de Protección

| Voltaje | Capacidad |
|---------|-----------|
| 120 V~ | Hasta 18.000 BTU |
| 220 V~ | Hasta 48.000 BTU |

## Diagrama de Conexión Monofásica

**Conexión para 120 V~ y 220 V~:**

```
F     T     N  →  Equipo a proteger
|     |     |
GSM-RF
|     |     |
F  T  N  S (Salida F)
```

**Leyenda:**
- F: Fase
- N: Neutro
- T: Tierra
- S: Salida Fase

## Diagrama de Conexión Bifásica

**Conexión trifásica (GSM-RB/MB/RF):**

```
F1    T    F2  →  Equipo a proteger
|     |     |
GSM-RF
|     |     |
F1 T F2 S (Salida F1)
```

**Leyenda:**
- F1: Fase 1
- F2: Fase 2
- T: Tierra
- S: Salida Fase 1
