# Regleta Puertos Smart USB A y C

## Descripción del Producto

Regleta de carga rápida e inteligente con puertos Smart USB A y C, diseñada para proteger y optimizar la carga de múltiples dispositivos electrónicos.

## Características Principales

- **Protección contra picos de alta energía**: 600 J
- **Optimización de vida útil de batería**: Tecnología Smart
- **Switch con protección de sobrecarga**: Incluido
- **Enchufe optimizado**: Diseño ergonómico para uso frecuente
- **4 tomas de salida**
- **3 puertos USB**: 2 USB A + 1 USB C

## Aplicaciones

- Celulares y Tablets
- TV y Cornetas
- Consolas de video juegos
- Electrónicos personales
- Módem y Routers
