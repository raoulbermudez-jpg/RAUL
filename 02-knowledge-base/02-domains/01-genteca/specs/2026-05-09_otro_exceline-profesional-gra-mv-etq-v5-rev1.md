# GRA-MV Relé Alternador

## Características Principales

- **Tipo de relé**: SPDT (Single Pole Double Throw)
- **Contacto común**: Borne 7
- **Carga máxima**: 3,5 A @ 250 V~
- **Dimensiones**: 50 mm x 52 mm

## Especificaciones Técnicas

| Parámetro | Valor |
|-----------|-------|
| Corriente nominal | 3,5 A @ 250 V~ |
| Tipo de contacto | SPDT |
| Voltaje de alimentación | 120/220 V~ |

## Distribución de Bornes

| Borne | Función |
|-------|---------|
| 1 | - |
| 2 | Señal control |
| 3 y 4 | Alimentación |
| 5 | - |
| 6 | Contacto de relé |
| 7 | Contacto común |
| 8 | Contacto de relé |

## Esquema de Conexión

El relé GRA-MV permite la alternancia automática entre dos equipos o bombas:

- **EQUIPO 1 / BOMBA 1**: Conectados a 120/220 V~
- **EQUIPO 2 / BOMBA 2**: Conectados a 120/220 V~
- **Modo AUTO**: Alternancia automática entre equipos/bombas
