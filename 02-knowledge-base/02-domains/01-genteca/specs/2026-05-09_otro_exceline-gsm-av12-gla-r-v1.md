# Aire Acondicionado de Ventana hasta 12.000 BTU/h - GSM-AV12

## Características

- Protección contra alto y bajo voltaje
- Protección contra parpadeos (flickers)
- Protección contra inestabilidad de red
- Protección rápida contra voltajes extremos
- LEDs indicadores de falla y estado del protector
- Temporizado inteligente de conexión
- Diseñado y programado para aires de ventana hasta 12.000 BTU/h
- Conexión a tierra

## Especificaciones Técnicas

| Parámetro | Valor |
|-----------|-------|
| Voltaje Nominal | 120 V~, 60 Hz |
| Rango de condición normal | 90-140 V~ |
| Desconexión programada | 0,02 a 2 segundos según intensidad de falla |
| Tiempo de desconexión ante parpadeos | 30 milisegundos |
| Temporizado inteligente de conexión | 3 minutos |
| Carga máxima | 12.000 BTU/h |
| Corriente máxima de compresor bloqueado | 60 A |
| Temperatura de operación | -5 a 55 ºC |
| Humedad relativa máxima de operación | 85% |

## Funcionamiento

En condiciones de voltaje normal, al enchufar el protector por primera vez a la energía eléctrica, este demorará al menos 3 minutos para energizar el aire acondicionado.

Si el voltaje se encuentra fuera del rango de condición normal, el protector desconectará el aire acondicionado e indicará el tipo de falla:
- LED rojo superior para voltaje alto
- LED rojo inferior para voltaje bajo

Si el voltaje presenta inestabilidad o variaciones extremas, el protector desconectará su aire de ventana de manera inmediata e indicará el tipo de falla: ambos LEDs rojos de manera intermitente para falla de frecuencia, alto o bajo voltaje con su respectivo LED.

Si la energía eléctrica se interrumpe de manera recurrente (parpadeos/flickers), el protector desconectará el aire acondicionado inmediatamente y solo lo energizará de nuevo una vez desaparecida la falla.

Una vez desaparecida cualquier falla que haya provocado la actuación del protector, este conectará nuevamente el aire acondicionado al culminar un temporizado inteligente que garantiza un apagado mínimo de 3 minutos.

## Instalación

1. Enchufe el protector al tomacorriente
2. Enchufe el equipo a proteger a la toma del protector Exceline

## Aplicación

Aire acondicionado de ventana con cualquier tipo de compresor (scroll, reciprocante, rotativo) hasta 12.000 BTU/h.
