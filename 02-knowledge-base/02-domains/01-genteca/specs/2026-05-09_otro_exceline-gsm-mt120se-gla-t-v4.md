# Protector de Voltaje Multifunción para Equipos Electrónicos

## Descripción del Producto

La única protección confiable en Venezuela para equipos electrónicos. Protector de voltaje multifunción que combina:

- Supresor de picos
- Filtro de línea
- Protección contra sobrecargas
- Regulación de altos y bajos voltaje

## Características Principales

### Funcionamiento Automático

- Se enciende y apaga automáticamente
- El interruptor se desconecta cuando el voltaje sube por encima de ~140 V
- Se desconecta cuando el voltaje cae por debajo de ~90 V
- Se reconecta automáticamente cuando se restaura la alimentación

### Indicadores

**Indicador de Voltaje Normal**
- Se ilumina cuando el voltaje de alimentación de energía al protector se encuentra dentro del rango permitido

**Indicador de Ciclo de Espera**
- Se ilumina cuando el usuario requiere montaje con tornillos

**Indicador de Falla de Voltaje / Sobrecarga**
- Se apaga automáticamente cuando suministra electricidad
- Para encender y apagar manualmente, presione el interruptor

## Instalación

### Montaje

- El protector dispone de dos orificios en su parte posterior para fijación a la pared
- El usuario requiere tornillos para el montaje
- Espere el tiempo de ciclo de espera antes de fijar a la pared

### Tomas Polarizadas

El equipo cuenta con seis tomas de alimentación. Cada una de las seis tomas está polarizada.

### Aplicaciones

- Audio
- Video
- Computación y Accesorios
