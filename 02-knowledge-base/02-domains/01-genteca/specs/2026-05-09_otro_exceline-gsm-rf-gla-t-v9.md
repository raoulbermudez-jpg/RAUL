# Protector de Voltaje para Compresores Monofásicos

## Descripción del Producto

Protector de voltaje diseñado para proporcionar protección contra:
- Altos y bajos voltaje
- Inestabilidad de voltaje
- Parpadeos (flickers)
- Variaciones extremas de voltaje

## Modelos Disponibles

### GSM-RF120B
- Capacidad: Hasta 18.000 BTU
- Voltaje: 120 V~
- Bornera para cables hasta AWG 8

### GSM-RF220B
- Capacidad: Hasta 48.000 BTU
- Voltaje: 220 V~
- Bornera para cables hasta AWG 8

## Características Principales

- **Nueva Bornera**: Diseño mejorado para cables hasta AWG 8
- **Fácil Instalación**: Configuración sencilla para el usuario
- Protección integral contra fluctuaciones de voltaje

## Diagrama Eléctrico

### Conexión GSM-RF120B (Monofásico)
- F: Fase
- N: Neutro
- T: Tierra
- S: Salida F

### Conexión GSM-RF220B (Bifásico)
- F1: Fase 1
- F2: Fase 2
- T: Tierra
- S: Salida F1
