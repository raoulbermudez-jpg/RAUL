# GIII+ - Relé de Protección Total para Motores

## Especificaciones Generales

### Versión 208/220V
- **Tensión nominal:** 208/220V~ 50/60Hz
- **Grado de protección:** IP20
- **Contacto:** 3A@240V~ / 1,5A@480V~
- **Aplicación:** Protección total para motores trifásicos

### Versión 440/480V
- **Tensión nominal:** 440/480V~ 50/60Hz
- **Grado de protección:** IP20
- **Contacto:** 3A@240V~ / 1,5A@480V~
- **Aplicación:** Protección total para motores trifásicos

## Características de Control

- **Monitoreo de fases:** VL1L2, VL2L3, VL3L1
- **Monitoreo de voltaje:** VUB
- **Monitoreo de corriente:** I1, I2, I3
- **Entrada auxiliar:** AUX
- **Salida:** CUB (Contacto Utilizable de Bloqueo)

## Funciones Operacionales

- Alerta visual/audible activable
- Encendido automático posible
- Control de rearme
- Ajuste de parámetros
- Selección de configuración
- Puertos de comunicación: GIO PORT

## Información de Diseño

- **Tamaño:** 167 mm (ancho) × 84 mm (alto)
- **Dimensiones de montaje:** 208 mm (altura total de panel)
- **Colores:** Pantone 151C / Cool Gray 9C (CMYK)
