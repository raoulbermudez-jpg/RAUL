# Protector de Voltaje para Televisores de Última Generación

## Especificaciones Técnicas

| Parámetro | Valor |
|-----------|-------|
| Modelo | GSM-EP120A |
| Voltaje Nominal | 120 V~ 50/60 Hz |
| Voltaje Máximo | 135 V~ |
| Voltaje Mínimo | 90 V~ |
| Carga Máxima | 20 A / 2400 W |
| Tiempo de Retardo | 25 s |

## Protección contra Sobretensiones

- **Supresor de Picos**: 400 V, 402 joules
  - Fase-Neutro (F-N)
  - Fase-Tierra (F-T)
  - Neutro-Tierra (N-T)

## Filtrado de Interferencias

- **Filtro EMI/RFI**: 15 dB
