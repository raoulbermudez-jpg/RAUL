# Propuesta de Estandarización de las Etiquetas Genius

## Información Técnica en Etiquetas Frontales

Las etiquetas frontales deben incluir los siguientes elementos técnicos:

- Código del producto
- Nombre del producto
- Sistema eléctrico (3 FASES)
- Voltaje
- Frecuencia
- Grado de protección IP
- Capacidad de los contactos o carga

## Formato de Etiqueta Modelo

La etiqueta modelo (set de etiquetas 6x1) contiene solo el logo Genius o Exceline. Se requiere desarrollar un formato sobre esta etiqueta que permita ubicar la información técnica para usar como referencia en la impresión durante el proceso de producción.

### Ubicación de Etiquetas

Las etiquetas se ubican en:
- Parte lateral del producto
- Parte posterior del producto
- Caja de cartón

## Información en Etiquetas Lateral/Posterior/Caja

Las etiquetas contienen la siguiente información:

- Nombre del producto
- Código del producto
- Versión de SW
- Serial del producto
- Código SKU del producto
- Características técnicas:
  - Sistema eléctrico (3 FASES)
  - Voltaje
  - Rango de Corriente (solo para GUCT+, GIII+, GSPT y GSPT-MV)
  - Modelo

## Elementos Identificadores

Los siguientes identificadores se mantienen sin cambios:

- Identificadores alrededor de la pantalla
- LEDs
- Teclas
- Puerto de comunicaciones
- Triángulo amarillo de alerta
