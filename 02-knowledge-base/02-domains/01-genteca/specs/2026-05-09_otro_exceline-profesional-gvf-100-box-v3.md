# GVF-100-HI Flotante Mecánico 1" para Manejo de Agua

## Descripción del Producto

Flotante mecánico de 1" diseñado para soluciones en manejo de agua para el hogar. Componente de fácil instalación que optimiza el llenado de tanques y depósitos de agua.

## Características Principales

- **Llena 3 veces más rápido**
- **Elimina vibración en tuberías**
- **Duradero, con componentes inoxidables**
- **100% libre de fugas**
- **Fácil instalación**

## Especificaciones Técnicas

| Parámetro | Especificación |
|-----------|----------------|
| Presión máxima | 150 PSI |
| Temperatura máxima del agua | 70°C |
| Conexión de entrada | Rosca NPT de 1" |
| Material | Nylon |
| Incluye | Filtro para retener partículas |

## Componentes

- Entrada (Rosca NPT)
- Flotante
- Prefiltro celulosa
- Salida interno
- Tanque o depósito

## Dimensiones

- Ancho: 45 cm
- Alto: 31,3 mm

## Principio de Funcionamiento

Para el correcto funcionamiento, se instala un filtro de celulosa que retiene el paso de partículas en las tuberías.

## Ventajas

- Confiabilidad
- Robustez
- Durabilidad
