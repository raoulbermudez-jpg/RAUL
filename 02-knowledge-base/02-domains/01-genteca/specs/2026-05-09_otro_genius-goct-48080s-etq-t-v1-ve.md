# GOCT-48080S - Relé de Protección Integral para Motores

## Descripción del Producto

Relé de protección integral para motores trifásicos con voltaje nominal de 440/480 VAC, frecuencia de 50/60 Hz.

## Características Principales

- **Puede encender automáticamente**
- **Contacto:** 1A @ 240V~ / 0.5A @ 480V~

## Funciones de Protección

- Conectado (ON)
- Temporizado (TC)
- Sobrecarga (OL)
- Fase Invertida (PR)
- Desbalance (UB)
- Pérdida de Fase (SP)
- Sobrevoltaje (OV)
- Bajo Voltaje (UV)

## Especificaciones Técnicas

| Parámetro | Valor |
|-----------|-------|
| Voltaje Nominal | 440/480 VAC |
| Frecuencia | 50/60 Hz |
| Fases | 3 |
| Rango FLA | 50-80 A |
| Rango TC | 40-250 S |
| Rango OL | 25-300 |
