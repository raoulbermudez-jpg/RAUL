ALIMENTACIÓN
TEMPERATURA
El GCT-S es un controlador digital para sistemas de refrigeración
que gestiona automáticamente:
Ciclo de refrigeración (compresor ON/OFF).
Protección del compresor con retardos de seguridad.
Monitoreo y alarma de temperatura.
Ciclo de descongelado programable y manual.

Menú de Usuario (SET – Hy)
Desbloquear el controlador.
Presionar SET mas de 3 segundos, ingresa a parámetro SET.
Usa ▲/▼ para cambiar entre SET y Hy.
Pulsar SET para ingresar al parámetro y ajustar su valor.

segundos, ingresa a parámetro HSE.

de 3 segundos.
Mantener presionado PRG durante 3 segundos para apagar el
