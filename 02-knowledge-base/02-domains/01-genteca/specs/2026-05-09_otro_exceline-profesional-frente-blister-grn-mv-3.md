# Relé de Nivel Profesional GRN-MV

## Características Principales

- **Sensibilidad Ajustable**
- **Multivoltaje**: 120 V~ / 220 V~
- **Dos Niveles de Control**: Máximo y Mínimo
- **Garantía**: 3 años

## Especificaciones Técnicas

- **Voltaje de Operación**: 120 V~ / 220 V~
- **Normas de Diseño**: IEC, UL y COVENIN

## Configuración de Relés

El equipo incluye:
- **Relé NC** (Normalmente Cerrado)
- **Relé NA** (Normalmente Abierto)
- **Electrodo Común**
- **Relé COMÚN**

## Diagrama de Conexión

Terminales de conexión:
- **Terminales 1-4**: Entrada de alimentación y control bajo (120/220V~)
- **Terminales 5-8**: Salida de relés (NC, COMÚN, NA)

## Aplicaciones

El relé GRN-MV es diseñado para control de nivel en:
- **Tanques**
- **Pozos**
