# Protector de Voltaje para Aires Acondicionados y Refrigeración - Alta Carga

## Características Principales

- Bornera para cables hasta AWG 8
- Fácil instalación
- Protección confiable para equipos de refrigeración y aires acondicionados
- Capacidad de protección hasta 18.000 BTU (monofásico) y hasta 48.000 BTU (trifásico)

## Diagrama Eléctrico - Configuración Monofásica (120V~ / 220V~)

```
    F ─────────────┐
                   ├─── Equipo a proteger
    N ─────────────┤
                   │
    T ─────────────┘
```

**Leyenda:**
- F: Fase
- N: Neutro
- T: Tierra
- S: Salida F

## Diagrama Eléctrico - Configuración Trifásica (120V~ / 220V~)

```
    F1 ────────────┐
                   ├─── Equipo a proteger
    F2 ────────────┤
                   │
    T  ────────────┘
```

**Leyenda:**
- F1: Fase 1
- F2: Fase 2
- T: Tierra
- S: Salida F1

## Especificaciones de Capacidad

| Configuración | Capacidad |
|---------------|-----------|
| Monofásico    | Hasta 18.000 BTU |
| Trifásico     | Hasta 48.000 BTU |
