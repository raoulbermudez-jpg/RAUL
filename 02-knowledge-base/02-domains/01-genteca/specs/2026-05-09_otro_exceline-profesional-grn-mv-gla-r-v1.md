# Relé de Nivel para Líquidos Conductores GRN-MV

## Descripción General

El GRN-MV es un relé (relevador) electrónico diseñado para controlar el vaciado o llenado de tanques con líquidos conductores. El control de los niveles se realiza mediante la conexión a electrodos de máximo nivel y/o de mínimo nivel (electrodos existentes en la instalación o provistos por el usuario).

El GRN-MV dispone de una salida de relé (relevador) para encender automáticamente y cada vez que se requiera al circuito arrancador de una bomba eléctrica.

El GRN-MV contiene dos indicadores luminosos que reportan cuando hay alimentación eléctrica y cuando el relé (relevador) de salida se encuentra activado.

## Características

- Conexión de 2 y 3 sondas, control en 1 y 2 niveles
- Amplia sensibilidad (2,5 a 300 kΩ)
- Multivoltaje 85 a 305 V~
- Conexiones tipo bornera
- Montaje tipo riel o superficie

## Aplicaciones

Sistemas de control de nivel de líquidos conductores, tales como:
- Agua potable
- Agua de pozo
- Agua de río
- Otros líquidos conductores

## Diagrama de Conexión

El dispositivo cuenta con 8 terminales de conexión distribuidos en dos grupos:
- Sonda de referencia
- Sonda de nivel máximo
- Sonda de nivel mínimo
- Aplicación (2 contactos de relé SPDT)

## Aviso de Instalación

Para instalar este dispositivo se podrían requerir herramientas y un personal técnico calificado.
