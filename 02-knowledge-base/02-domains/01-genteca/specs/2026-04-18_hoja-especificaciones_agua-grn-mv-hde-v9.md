# El GRN-MV de Exceline Profesional  es un relé (relevador)

El GRN-MV de Exceline Profesional  es un relé (relevador)
electrónico, diseñado para el control de nivel de líquidos
conductores.
Su funcionamiento se basa en la lectura de las sondas instaladas,
permitiendo un control preciso y confiable. Este dispositivo
admite dos configuraciones:
• Aplicación de 1 Nivel: Utiliza dos sondas (una de referencia
y otra para el nivel a controlar).
• Aplicación de 2 Niveles: Requiere tres sondas (una de
referencia, otra para el nivel bajo y la tercera para el nivel alto).
El control se realiza mediante la actuación del relé interno, de
salidas tipo SPDT.
El GRN-MV de Exceline Profesional incorpora un sistema
inteligente de memoria de tarea, que permite mantener la
continuidad del servicio incluso después de una interrupción
eléctrica. Si el equipo se encontraba en proceso de vaciado o
llenado al momento del corte, retoma automáticamente la última
función al restablecerse la energía, sin requerir intervención del
usuario.
El usuario puede ajustar la sensibilidad mediante una perilla,
ajustando el equipo según la conductividad del líquido utilizado.
El GRN-MV de Exceline Profesional dispone de dos indicadores
luminosos que reportan: uno la presencia de alimentación eléctrica
(ENCENDIDO) y otro, que la etapa de salida de control esta
conectada (CONECTADO).
Diseñado y verificado bajo las normas IEC de compatibilidad
electromagnetica y construido con materiales de altísima calidad,
el GRN-MV de Exceline Profesional  es la mejor alternativa
para cualquier aplicación profesional.
