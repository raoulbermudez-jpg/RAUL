# GVF-100-HI - Válvula Flotante Mecánica 1" para el Hogar

## Características Principales

- **Llena 3 veces más rápido**
- **Elimina vibración en tuberías**
- **Duradero, con componentes inoxidables**
- **100% libre de fugas**
- **Fácil instalación**

## Especificaciones Técnicas

| Parámetro | Valor |
|-----------|-------|
| Presión Máxima | 150 PSI |
| Temperatura Máxima del Agua | 70°C |
| Tipo de Rosca | NPT (entrada) |
| Conexión | Mecánico 1" |
| Material | Nylon |
| Incluye | Filtro para retener partículas (prefiltro) |

## Descripción del Producto

Válvula flotante mecánica de 1" diseñada para soluciones en manejo de agua para el hogar. Instala un filtro de celulosa para el funcionamiento correcto.

## Componentes

- Entrada de agua (rosca NPT)
- Flotante
- Prefiltro
- Salida interna
- Tanque o depósito
- Máximo nivel del flotante
