# Protector de Voltaje para Compresores Monofásicos

## Características principales

- **Capacidad**: Hasta 48.000 BTU
- **Voltaje nominal**: 220 V~
- **Tipo**: Monofásico

## Protecciones

El dispositivo protege contra:
- Altos y bajos voltaje
- Inestabilidad de voltaje
- Variaciones extremas
- Parpadeos

## Diagrama Eléctrico

### Conexiones

**Entrada:**
- F1: FASE 1
- F2: FASE 2
- T: TIERRA

**Salida:**
- F1: FASE 1
- F2: FASE 2
- S: SALIDA
- T: TIERRA

### Componentes del circuito

- MCP: Protección principal
- OOR: Detección de rango
- OMT: Monitoreo de voltaje
- EPC: Control de energía
- FRT: Relé de transferencia
- Á: Accionamiento
- OER: Protección externa
- SS: Sensor de sobrevoltaje
- D: Relé de desconexión
- IOC: Control de corriente
- E: Entrada
- RO: Relé de salida
- VOES: Sensor de voltaje
- LSTAJE: Sistema de bloqueo
