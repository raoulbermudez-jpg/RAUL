# Protector de Voltaje para Equipos Telefónicos GSM-MT120T

## Especificaciones Técnicas

| Parámetro | Valor |
|-----------|-------|
| **Modelo** | GSM-MT120T |
| **Voltaje Nominal** | 120 V~, 50/60Hz |
| **Voltaje Máximo** | 140 V~ |
| **Voltaje Mínimo** | 90 V~ |
| **Carga Máxima** | 15A, 1800W |
| **Tiempo de Retardo** | 30 segundos |

## Protección contra Picos

- **Supresor de Picos en la Alimentación**: 400 V, 606 Joules (F-N, F-T, N-T)
- **Supresor de Picos Telefónicos**: 198 joules

## Filtrado

- **Filtro de Ruidos EMI/RFI**: 15dB
