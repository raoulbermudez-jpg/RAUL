# Relé de Nivel VM-NRG para Líquidos Conductores Multivoltaje 120/220 V~

## Descripción General

El VM-NRG es un relé electrónico diseñado para controlar el nivel de tanques llenos o vaciados con líquidos conductores. Es un producto original único que realiza el control de niveles mediante conexión a electrodos de nivel máximo y/o nivel mínimo.

## Características

- Dos indicadores luminosos que reportan cuando hay alimentación eléctrica y cuando el relé se encuentra activado
- Salida de relé disponible para encender automáticamente (cada vez que sea requerido) una bomba eléctrica o arrancador de circuito
- Robustez, calidad y confiabilidad
- Garantía válida de tres años desde la fecha de adquisición

## Aplicaciones

- Control de nivel único para tanques de almacenamiento
- Conexión para controlar dos puntos de nivel (máximo y mínimo)
- Ajuste de descarga o depósito de líquidos en tanques de fabricación de 10K
- Fijación de sensibilidad de bombas eléctricas y tableros que suministren electrodos de encendido y apagado
