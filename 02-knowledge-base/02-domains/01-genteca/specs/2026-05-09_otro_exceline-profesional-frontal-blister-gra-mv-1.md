# Relé Alternador Profesional para Sistemas Hidroneumáticos

## Características Principales

- **Multivoltaje**: 120 V~ / 220 V~
- **Garantía**: 3 años
- **Normas de diseño**: IEC, UL y COVENIN

## Aplicación

Relé alternador profesional diseñado para control de sistemas hidroneumáticos con capacidad de gestionar dos bombas.

## Diagrama de Conexión

| Componente | Bornes |
|-----------|--------|
| Presostato | P |
| Entrada de alimentación | 120/220 V~ |
| Bobina Contactor Bomba 1 | 1, 2, 3, 4 |
| Bobina Contactor Bomba 2 | 5, 6, 7, 8 |

## Nota Importante

Para una correcta operación, el relé debe permanecer energizado en los bornes 3 y 4.
