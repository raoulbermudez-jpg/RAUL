# GVF Flotante Mecánico - Soluciones en Manejo de Agua para el Hogar

## Modelos Disponibles

- **GVF-150-HI** - Rosca 1½"
- **GVF-200-HI** - Rosca 2"

## Características Técnicas

- **Presión**: 150 PSI
- **Temperatura máxima en el agua**: 70°C
- **Rosca de entrada**: NPT
- **Material**: Nylon
- **Incluye**: Filtro para retener el paso de partículas

## Ventajas Principales

- Llena 3 veces más rápido
- Elimina vibración en tuberías
- Duradero, con componentes inoxidables
- 100% libre de fugas

## Instalación

**IMPORTANTE**: Para maximizar la vida útil y correcto desempeño de este flotante, coloque un prefiltro de celulosa antes del mismo, tal como indica el diagrama de instalación.

### Diagrama de Instalación

```
Entrada Flujo de agua → Prefiltro → Flotante → Salida
                                    (Máximo nivel del flotante interno)
                                           ↓
                                    Tanque o depósito
```
