# Etiquetas con Toroide - Conexiones Externas

## Configuración de Entradas y Salidas

### Entrada Digital
- Utiliza GIO PLUG (Opcional) para comunicación serie con otros dispositivos

### Toroide Externo
Modelos disponibles:
- SX11/k
- PH22/L
- SX2/2l

## Estados de Operación

### Estado Normal
- Relé de control: Conectado
- Relé auxiliar: Abierto

### Estado de Disparo (Tripped)
- Relé de control: Abierto
- Relé auxiliar: Conectado

## Comunicación Opcional
Se puede utilizar GIO PLUG para comunicación serie con otros dispositivos mediante el circuito de inicio de línea.

## Notas de Aplicación
- Las conexiones de toroide externo deben realizarse según el diagrama indicado
- La opción de temperatura debe considerarse en la selección del modelo apropiado
