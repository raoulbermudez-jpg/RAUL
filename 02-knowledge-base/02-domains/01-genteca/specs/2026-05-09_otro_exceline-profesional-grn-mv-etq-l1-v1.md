# Etiqueta técnica GRN-MV

## Especificaciones físicas

- **Ancho:** 50 mm
- **Alto:** 26 mm
- **Color:** Negro (Black)

## Configuraciones de aplicación

### Aplicación 1 nivel
- **Altura de montaje:** 62 mm
- **Conexión:** Sonda de referencia (terminales 5, 6, 7, 8)
- **Sensor:** Sonda nivel mínimo (terminales 1, 2, 3, 4)

### Aplicación 2 niveles
- **Configuración lateral 1**
- **Conexión principal:** Sonda de referencia (terminales 5, 6, 7, 8)
- **Sensores:** 
  - Sonda nivel máximo (terminales 1, 2, 3, 4)
  - Sonda nivel mínimo (terminales 1, 2, 3, 4)

## Diagrama de conexión

El producto incluye esquema de conexión para ambas configuraciones (1 y 2 niveles) con identificación clara de terminales de sonda de referencia y sensores de nivel.
