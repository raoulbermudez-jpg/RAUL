# Protector de Voltaje para Compresores Monofásicos

## Descripción del Producto

Protector de voltaje diseñado para proporcionar protección contra:
- Altos y bajos voltaje
- Inestabilidad de voltaje
- Parpadeos (Flicker)
- Variaciones extremas

## Modelos Disponibles

- **GSM-RF120B**: Para 120 V~
- **GSM-RF220B**: Para 220 V~

## Diagrama Eléctrico

### GSM-RF120B (Monofásico 120V)

**Conexión:**
- F: FASE
- N: NEUTRO
- T: TIERRA
- S: SALIDA F

**Conexión al equipo a proteger:**
- Entrada: F, N, T
- Salida: S, N, T

### GSM-RF220B (Monofásico 220V)

**Conexión:**
- F1: FASE 1
- F2: FASE 2
- T: TIERRA
- S: SALIDA F1

**Conexión al equipo a proteger:**
- Entrada: F1, T, F2
- Salida: S, T, F2
