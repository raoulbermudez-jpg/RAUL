# Protector de Voltaje para Aires Acondicionados y Refrigeración

## Características Principales

- **Nuevo**: Bornera para cable hasta calibre AWG 8
- **Instalación más cómoda**
- Protección contra:
  - Altos y bajos voltaje
  - Inestabilidad
  - Parpadeos
  - Variaciones extremas

## Modelos Disponibles

### GSM-R120B
- Voltaje: 120 V~
- Capacidad: Hasta 18.000 BTU
- Conexión monofásica (F-N-T)

### GSM-R220B
- Voltaje: 220 V~
- Capacidad: Hasta 48.000 BTU
- Conexión bifásica (F1-F2-T)

## Diagrama Eléctrico

### GSM-R120B (Monofásico)

**Leyenda:**
- F: Fase
- N: Neutro
- T: Tierra
- S: Salida F

### GSM-R220B (Bifásico)

**Leyenda:**
- F1: Fase 1
- F2: Fase 2
- T: Tierra
- S: Salida F1

## Especificaciones de Bornera

- Compatible con cables hasta calibre AWG 8
- Conexión segura para instalación en aplicaciones de alta carga
