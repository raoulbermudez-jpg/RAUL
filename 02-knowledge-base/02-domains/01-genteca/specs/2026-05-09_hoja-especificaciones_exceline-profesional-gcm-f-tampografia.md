# GCM-F Tampografía

## Especificaciones Técnicas del Producto GCM-F250-100

### Características Generales
- **Tipo**: MCCB (Interruptor Automático de Caja Moldeada)
- **Corriente Nominal**: In = 100 A
- **Frecuencia Nominal**: 50~60 Hz
- **Dimensión de etiqueta de impresión**: 33,5 mm

### Voltajes y Corrientes de Operación

| Parámetro | Especificación |
|-----------|----------------|
| Ue (Voltaje de Servicio) | AC 690V |
| Ui (Voltaje de Impulso) | AC 690V |
| Uimp (Voltaje de Impulso de Sobretensión) | 8 kV |
| Ii (Corriente de Impulso) | 10In |

### Capacidad de Cortocircuito (Icu/Ics)

**A AC 690V**: 10/5 kA

**A AC 400V**: 50/35 kA

**A AC 230V**: 100/50 kA

### Características de Conexión
- **Terminales**: Compatibles con conexiones de Aluminio (AI) y Cobre (Cu)

### Especificaciones de Tampografía
Se definen tres pantones de impresión para las etiquetas de cobertura (Cover printing label 1 y 2):

- **Pantone A**: 7548 C
- **Pantone B**: 151 C
- **Pantone C**: 151 C
