# GVF-100-HI - Soluciones Mecánico 1" en Manejo de Agua Flotante para el Hogar

## Características

- **Llena 3 veces más rápido**
- **Elimina vibración en tuberías**
- **Duradero, con componentes inoxidables**
- **100% libre de fugas**
- Incluye filtro para retener el paso de partículas
- Rosca NPT de entrada

## Especificaciones Técnicas

| Parámetro | Valor |
|-----------|-------|
| Presión máxima | 150 PSI |
| Temperatura máxima en el agua | 70°C |
| Material | Nylon |
| Conexión | Mecánico 1" |

## Instalación

Para maximizar la vida útil y correcto desempeño de este flotante, coloque un pre filtro de celulosa de entrada antes del mismo tal como indica el diagrama de instalación.

**Componentes del sistema:**
- Entrada Flujo de agua
- Prefiltro
- Flotante
- Máximo nivel del flotante
- Salida
- Tanque o depósito interno
