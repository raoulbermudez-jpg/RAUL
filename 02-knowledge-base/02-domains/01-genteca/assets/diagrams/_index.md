# Assets / Diagrams — Genteca

Diagramas eléctricos de instalación y conexión: unifilares y trifilares.

## Subcarpetas

- `Unifilares/` — diagramas unifilares (single-line diagrams)
- `Trifilares/` — diagramas de conexión trifásica

**Convención:** `<CÓDIGO-PRODUCTO>_<tipo>_v<N>.<ext>`  
Ejemplo: `GSM-MB_triflar_v1.png`

## Uso

Renzo lee estos diagramas directamente (PNG/JPG/PDF) para generar guías de instalación y documentación técnica de campo.
