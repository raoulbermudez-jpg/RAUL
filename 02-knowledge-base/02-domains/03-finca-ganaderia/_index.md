# Finca-ganadería — Dominio

**Estado:** Pendiente de activación. Carpetas skeleton creadas en 2026-04-25. No iniciar Fase 3 sin luz verde explícita del Owner.

**Alcance:** Operaciones ganaderas de Raoul Bermudez. Incluye gestión de finca, rotación de potreros, sanidad animal, producción, logística y comercialización. Potencial de proyectos de consultoría y modelos de simulación agropecuaria.

**Distinción con otros dominios:**
- **Finca-ganadería ≠ Teca/Teak4Food:** Teak4Food es la marca comercial (madera teca, producción forestal); Finca-ganadería es la operación pecuaria.
- **Finca-ganadería ≠ Plenus:** Plenus es alimentos funcionales para el consumidor final; Finca es la operación primaria de producción.
