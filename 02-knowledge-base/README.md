# 02-knowledge-base/

LLM Wiki compilada. Memoria acumulativa del sistema Raul.

## Subcarpetas

- `01-foundations/` — Metodología, filosofía local-first, patrones de uso de LLM.
- `02-domains/` — Conocimiento por dominio (Genteca, Finca, Plenus, Teca, marca personal). Cada dominio puede tener `wiki/` (artículos), `specs/` (fichas técnicas) y `assets/` (visuales).
- `03-cross-cutting/` — Temas transversales que no pertenecen a un dominio único. Incluye `assets/` para material realmente genérico.
- `04-sops-and-playbooks/` — SOPs generales reutilizables (no content supply chain, eso vive en `04-system/`).
- `05-glossary-and-tables/` — Glosarios y tablas de referencia.

## Regla

Aquí solo entran documentos **compilados y curados**, no dumps crudos. Todo es Markdown. Cada dominio tiene `_index.md`.
